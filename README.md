# 📝 Smart Complaint Box

A simple and effective complaint management system built using Flask and PostgreSQL.

---

## 🌐 Live Demo
https://smart-complaint-box-gr0v.onrender.com

---

## 🚀 Features

### 👤 User
- Submit complaints easily
- Simple and clean interface

### 🔐 Admin
- Secure login
- View all complaints
- Delete complaints
- Logout system

---

## 🛠 Tech Stack

- Python (Flask)
- PostgreSQL
- HTML (Templates)
- Gunicorn
- Render (Deployment)
